8086 Instruction Timing Analyzer

Overview:
An Assembly Language project built using EMU8086 that analyzes 8086 instructions and code blocks to calculate clock cycles based on microprocessor timing rules.

Features:
- Single instruction analysis  
- Code block analysis  
- Timing reference table  
- Memory vs register cycle detection  
- Menu-driven interface  

Tools Used:
- 8086 Assembly Language  
- EMU8086 Emulator  
- INT 21h, INT 10h, INT 16h  

Key Concepts:
- Instruction timing calculation  
- Effective Address handling  
- Register and memory operations  
- Low-level system programming  

Conclusion:
This project demonstrates how 8086 instructions execute at the microprocessor level by analyzing their clock cycle consumption in an interactive way.
